# Agente_Triagem_Medica.ipynb


Este repositório contém a implementação de um Agente de Triagem Médica inteligente, desenvolvido durante a Imersão Tech: Agentes de IA com Python da WoMakers Code.

O projeto demonstra como utilizar modelos de linguagem (LLMs) integrados ao framework LangChain para criar fluxos de triagem automatizados, focados em analisar sintomas e fornecer orientações preliminares de forma segura e estruturada.

 Sobre o Projeto
O objetivo deste agente é atuar como um assistente de triagem que coleta informações de pacientes, analisa os sintomas descritos e fornece uma classificação inicial de prioridade, seguindo diretrizes de Engenharia de IA. E o outro agente de IA é para coleta de buscas de informações na internet.

Principais conceitos aplicados:
Agentes Autônomos: Estrutura de observação, raciocínio e ação.
LangChain: Orquestração do fluxo de trabalho do agente.
Prompt Engineering: Técnicas de estruturação de prompts para garantir respostas precisas e seguras.
Segurança: Implementação de boas práticas para manipulação de chaves de API.

Tecnologias Utilizadas
Python
Google Gemini API
LangChain
Google Colab

 Como executar
Este projeto foi construído para ser executado no Google Colab. Para rodá-lo localmente ou no seu próprio Colab, siga os passos abaixo:
Obtenha sua API Key: Acesse o Google AI Studio e gere uma chave de API para o Gemini.
Configuração de Secrets:
No seu Google Colab, clique no ícone de chave (Secrets) no menu lateral esquerdo.
Adicione um novo segredo com o nome: GOOGLE_API_KEY.
Cole o valor da sua chave no campo correspondente.
Execução: Siga a ordem das células no notebook, garantindo que as bibliotecas necessárias estejam instaladas.
Nota de Segurança: Nunca publique o valor real da sua API Key diretamente no código. Utilize sempre o gerenciador de segredos do ambiente.
